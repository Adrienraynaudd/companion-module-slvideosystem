# SLVideoSystem

 This Companion module connects to the SLVideoSystem TCP server on port 7891.

## Configuration

Set the SLVideoSystem host address and the grid size expected by SLVideoSystem.

The TCP port is fixed to 7891 and the hardware ID authentication handshake is handled automatically by the module.

The module sends the grid size when the connection opens and sends it once again shortly after connection to help resynchronize after an SLVideoSystem restart.

## Recommended usage

Companion does not expose the real screen position of a button to the module, so the module cannot infer that a button is physically placed at 0/3, 2/3, and so on.

Use the presets from the `SLVideo Grid` category.

Each preset adds both of these at once:

- the `Trigger shortcut` action
- the `Shortcut visual state` advanced feedback

The preset does not save a fixed `shortcutId`. It stores the SLVideo row and column, and the module resolves the current shortcut for that grid slot from the latest shortcut list received from SLVideoSystem.

If you add the action or the feedback manually, either pick a specific SLVideo shortcut from the dropdown or leave the shortcut empty and set the row and column values yourself.

## How it works

The module:

- sends the configured grid size to the server
- downloads the JSON shortcut lists generated by SLVideoSystem
- maps each preset to the shortcut currently assigned to that grid slot
