# SLVideoSystem Companion Module

Companion module for controlling SLVideoSystem over TCP.

## Features

- Connects to the SLVideoSystem TCP server on port 7891
- Synchronizes the remote shortcut grid and exposes Companion presets
- Triggers shortcuts by grid position or by explicit shortcut selection
- Mirrors shortcut colors and icons through advanced button feedback

## Requirements

- Bitfocus Companion with module support for Node 22 runtimes
- Network access from Companion to the SLVideoSystem host on TCP port 7891

## Configuration

Configure the module in Companion with:

- the SLVideoSystem host or IP address
- the expected grid width and height

The TCP port is fixed to 7891. Authentication is handled automatically with the hardware ID generated by the Companion instance.

## Usage

The recommended workflow is to use the presets generated by the module. Each preset binds a Companion button to one SLVideoSystem grid position and applies the matching advanced feedback.

If you build buttons manually, either:

- select a specific SLVideoSystem shortcut from the dropdown
- or leave the shortcut empty and set the SLVideo row and column values
